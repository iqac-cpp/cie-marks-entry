# CIE Marks Entry Portal - GitHub Pages Direct Open

Institution: GDC Cheepurupalli  
Project: Continuous Internal Evaluation Marks Entry Portal

## Purpose

This GitHub Pages version directly opens the final CIE Web App.

## Files

- `index.html` - GitHub Pages landing file.
- `README.md` - Setup notes.

## Upload Instructions

1. Upload `index.html` and `README.md` to the GitHub repository root.
2. Go to repository `Settings`.
3. Open `Pages`.
4. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Click `Save`.
6. Wait 1 to 3 minutes.
7. Open the GitHub Pages URL.

## Notes

- The GitHub Pages link will directly open the CIE Web App.
- No separate Apps Script link is displayed on the page.
- The Apps Script Web App URL can still be shared directly from Google Sheets/Apps Script whenever needed.
